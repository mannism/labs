CREATIVE CODING — LABS V2 CANVAS ANIMATIONS

Labs v2 uses a "Speculative Interface" design direction — light ground (#F0F2F5), chartreuse accent, and a layer of creative animations that respond to user interaction and page state. These are not decorative. They are the interface language.

The animations:
- Ghost Type — text that appears to type itself with spectral persistence, characters fading in and decaying. Built with requestAnimationFrame and React hooks.
- System Boot — a startup sequence animation simulating a system initialisation, setting the tone before content loads.
- Signal Field — a responsive particle field that reacts to cursor proximity and movement. Built with HTML Canvas API.
- Datamosh — glitch-aesthetic visual distortion applied to transitions and media elements. Built with Framer Motion.
- Proximity Pulse — radial pulse effects triggered by cursor position, creating a sense of the interface responding to presence.

The creative coding layer uses a mix of approaches — Canvas API for particle systems, Framer Motion for motion design and transitions, and React hooks with requestAnimationFrame for text-based effects. All developed through Claude Code, the same agentic workflow used for everything else on Labs.

This sits at the intersection of creative direction and engineering — the kind of work that usually requires a creative technologist and a developer working together. Here, I am both, with Claude Code as the build partner.
